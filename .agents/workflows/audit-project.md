---
description: Executa auditoria completa de um sistema usando o QA Master Auditor
---

Quando o usuário chamar `/audit-project`:

1. Ativar o skill `qa-master-auditor`
2. Analisar todo o contexto fornecido (código, descrição, logs)
3. Executar:
   - análise de fluxo
   - análise de integração
   - análise de dados
   - análise de falhas
4. Gerar output no formato padrão do auditor
