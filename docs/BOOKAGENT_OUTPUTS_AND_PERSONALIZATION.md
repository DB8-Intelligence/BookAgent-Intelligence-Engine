# BookAgent Intelligence Engine — Outputs e Personalização v1.0

## 1. Outputs Sociais

### 1.1 Reels

| Atributo | Especificação |
|---|---|
| **Formato** | MP4 |
| **Aspect ratio** | 9:16 (vertical) |
| **Duração** | 15s, 30s, 60s ou 90s (configurável) |
| **Resolução** | 1080x1920 |
| **FPS** | 30 |
| **Codec** | H.264 |
| **Canal** | Instagram Reels, TikTok, YouTube Shorts |

**Estrutura típica**:
1. Hook visual (2-3s) — imagem hero com texto impactante
2. Sequência de destaques (80% da duração) — imagens com transições + texto overlay
3. CTA final (3-5s) — dados de contato + logo

**Fontes necessárias**: hero (obrigatório), lifestyle ou infraestrutura (2+ imagens), CTA

## 5. Regras de CTA

### Onde o CTA aparece em cada output

| Output | Posição do CTA | Formato |
|---|---|---|
| Reel | Últimos 3-5 segundos | Tela com nome + WhatsApp + Instagram |
| Story | Último card | Card completo com dados |
| Carrossel | Último slide | Slide com todos os dados + logo |
| Post | Rodapé da imagem | Nome + WhatsApp abaixo da imagem |
| Blog | Final do artigo + sidebar | Bloco CTA com formulário inline |
| Landing page | Hero + final da página | Botão flutuante + formulário |
| Vídeo longo | Abertura (logo) + final | Tela completa de contato |
| Apresentação | Primeiro e último slide | Dados completos |
| Áudio | Final da narração | Menção verbal dos dados |

### Regras de Formatação

- **WhatsApp**: Sempre exibir como link clicável (wa.me/5511999999999)
- **Instagram**: Sempre com @ (ex: @corretor_nome)
- **Site**: Sempre sem "https://" no visual (ex: www.site.com.br)
- **Nome**: Sem abreviações
- **Região**: Usado para contextualização (ex: "Especialista na região do Tatuapé")

### Fallback (Dados Ausentes)

- Se nenhum dado de CTA fornecido → não exibir bloco de CTA (output genérico)
- Se apenas WhatsApp → CTA simplificado ("Fale com consultor: (11) 99999-9999")
- Se todos os dados → CTA completo com layout rico

## 6. Diferença Entre Vídeo Curto e Vídeo Longo

| Atributo | Vídeo Curto | Vídeo Longo |
|---|---|---|
| **Duração** | Até 120 segundos | Acima de 120 segundos |
| **Objetivo** | Engajamento, descoberta, viralização | Informação completa, decisão de compra |
| **Canal** | Instagram, TikTok, YouTube Shorts | WhatsApp, Dashboard, YouTube |
| **Ritmo** | Rápido, cortes frequentes (2-3s por cena) | Moderado, cenas mais longas (5-10s) |
| **Texto** | Frases curtas, 1-2 palavras por tela | Parágrafos curtos, dados detalhados |
| **Narração** | Opcional (música predomina) | Recomendada (voz descritiva) |
| **CTA** | Simples (1 dado) | Completo (todos os dados) |
| **Compressão** | Qualidade alta (sem limite rígido) | Otimizada para 16MB (WhatsApp) |
| **Resolução** | 1080x1920 (9:16) | 720x1280 ou 1280x720 |
